# m4

マクロプロセッサ

## FizzBuzz

`bin/fizzbuzz` が対話環境の起動用シェルスクリプトです。  
`src/fizzbuzz.m4` を読み込ませ、警告/デバッグオプションを自動で設定してくれます。

```bash
$ ./bin/fizzbuzz
fizzbuzz(1, 15)
1, 2, 3 <- Fizz, 4, 5 <- Buzz, 6 <- Fizz, 7, 8, 9 <- Fizz, 10 <- Buzz, 11, 12 <- Fizz, 13, 14, 15 <- FizzBuzz
```

## テスト

```bash
$ ./test.bash
```
